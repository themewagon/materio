## Getting Started

1. Clone from Github 
```
git clone https://github.com/themewagon/materio.git
```
2. Navigate to the working version
```
cd typescript-version/

or, cd javascript-version/
```
3. Install dependencies
```
npm install --legacy-peer-deps
```
